This repository contains code and data supporting the paper *“Structural changes in autism reflect atypical brain network organization and phenotypical heterogeneity: a hybrid deep network approach”* (https://doi.org/10.1101/2025.11.02.686152).

## Repository Structure

data/ – data used for analysis

figures/ – plots and visualizations

nn code/ – code for deep learning models

saliency/ – saliency pipeline and raw saliency data
